# Менеджер паролей

Простой менеджер паролей на Python с графическим интерфейсом. Позволяет хранить пароли и данные с возможностью поиска по описанию.

## Функционал

* При запуске `main.py` открывается окно для создания или ввода мастер-пароля.
* После входа доступны две основные кнопки:

  * **Создать** — добавить новый пароль.
  * **Прочитать** — просмотреть сохранённые пароли.
* Форма поиска позволяет искать записи по описанию. Например, если описание `ВКонтакте`, можно ввести `вкон` и найти запись.

## Требования

* Python 3.12+
* UV для управления зависимостями ([https://uv.io](https://uv.io))

## Установка окружения

1. Клонируйте репозиторий:

```bash
git clone <URL_репозитория>
cd <название_репозитория>
```

2. Установите UV (если не установлен):

```bash
python -m pip install uv
```

3. Установите зависимости через UV:

```bash
uv install
```

4. Запустите приложение:

```bash
python main.py
```

## Примечания

* Мастер-пароль используется для защиты всех ваших записей.
* Поиск по описанию нечувствителен к регистру и поддерживает частичные совпадения.
* Убедитесь, что зависимости установлены через UV перед запуском.
