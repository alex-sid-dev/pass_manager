# Менеджер паролей

Простой менеджер паролей на **Python** с графическим интерфейсом. Приложение позволяет безопасно хранить пароли и
связанные с ними данные, а также выполнять поиск по описанию.

---

## Функционал

* При запуске `main.py` открывается окно для создания или ввода **мастер-пароля**.

* После успешного входа доступны две основные кнопки:

    * **Создать** — добавить новую запись с паролем.
    * **Прочитать** — просмотреть сохранённые пароли.

* Реализован поиск по описанию:

    * поиск нечувствителен к регистру;
    * поддерживаются частичные совпадения (например, по запросу `вкон` будет найдена запись `ВКонтакте`).

---

## Требования

* Python **3.12+**
* **UV** — менеджер зависимостей ([https://uv.io](https://uv.io))

---

## Установка и настройка окружения

### 1. Создание папки для менеджера паролей

Linux / macOS (bash)

```bash
mkdir -p ~/projects/pass_manager
cd ~/projects/pass_manager
```

Windows (PowerShell)

```bash
mkdir $HOME\projects\pass_manager
cd $HOME\projects\pass_manager
```

### 2. Клонирование репозитория

```bash
git clone https://github.com/alex-sid-dev/pass_manager.git
```

### 3. Установка UV (если не установлен)

```bash
python -m pip install uv
```

### 4. Установка зависимостей

```bash
uv sync
```

## Создание ярлыка для запуска

### Windows

1. Создайте ярлык на рабочем столе.
2. В поле **«Объект»** укажите путь:

```text
"C:\Путь_к_проекту\pass_manager\.venv\Scripts\pythonw.exe" "C:\Путь_к_проекту\pass_manager\main.py"
```

3. Сохраните ярлык — после этого приложение можно запускать двойным щелчком.

---

### Linux

#### Вариант 1: запуск через терминал

```bash
cd /путь/к/проекту/pass_manager
uv run python main.py
```

#### Вариант 2: ярлык (.desktop файл)

1. Создайте файл ярлыка:

```bash
nano ~/.local/share/applications/pass_manager.desktop
```

2. Вставьте следующее содержимое (измените пути при необходимости):

```ini
[Desktop Entry]
Name = Password Manager
Comment = Простой менеджер паролей
Exec = /путь/к/проекту/pass_manager/.venv/bin/python /путь/к/проекту/pass_manager/main.py
Icon = dialog-password
Terminal = false
Type = Application
Categories = Utility;Security;
```

3. Сделайте файл исполняемым:

```bash
chmod +x ~/.local/share/applications/pass_manager.desktop
```

4. После этого приложение появится в меню приложений вашей системы.

---

## Примечания

* Мастер-пароль используется для защиты всех сохранённых записей.
* Без знания мастер-пароля доступ к данным невозможен.
* Перед запуском убедитесь, что все зависимости установлены через **UV**.
